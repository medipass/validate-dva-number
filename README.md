# validate-dva-number

> Validates a DVA card number


## Install

```
$ yarn add validate-dva-number
```

or NPM:

```
$ npm install validate-dva-number
```

## Usage

```js
const validateDvaNumber = require('validate-dva-number');

validateDvaNumber('NB123456');
//=> true
```


## API

### validateDvaNumber(input)

#### input

Type: `string`

The DVA card number
